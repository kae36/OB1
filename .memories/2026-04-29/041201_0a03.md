## Explored Open Brain Dashboards

Compared two dashboard options for Open Brain:

### Findings
- **open-brain-dashboard (SvelteKit):** Lighter weight, 4-page UI for capture/search/browse/stats. Uses SvelteKit + Supabase auth. Calls MCP directly.
- **open-brain-dashboard-next (Next.js):** Feature-rich with 9 pages including workflow kanban board, audit, duplicates detection. Requires `open-brain-rest` REST API gateway deployed separately.

### Key Clarification
- **OpenRouter** ≠ `open-brain-rest`. OpenRouter is an AI API gateway service for embeddings (used during core setup). `open-brain-rest` is a Supabase Edge Function that provides REST endpoints for the Next.js dashboard.

### Recommendation
- Start with **Next.js** for full feature set (assumes REST API gateway is available)
- Use **SvelteKit** if wanting lightweight quick setup without deploying extra infrastructure"
