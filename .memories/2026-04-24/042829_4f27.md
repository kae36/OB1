## Open Brain — Session: Updating & Exploring Skills

Session focused on understanding the Open Brain repo workflow after initial setup was completed last session.

## Key Learnings

- **How to get new community content:** `git pull` (no args) works the same as `git pull origin main` for a standard clone — pulls from tracked remote/branch
- **No auto-install:** New extensions, recipes, schemas, and skills are opt-in. Each lives in its own subfolder with a README. You apply what you want manually:
  - `schemas/` → run SQL in Supabase
  - `extensions/` → deploy Edge Function per README
  - `recipes/` → deploy Edge Function or configure connector
  - `skills/` → copy prompt/skill into AI client
  - `integrations/` → deploy Edge Function, add connector URL in Claude Desktop
