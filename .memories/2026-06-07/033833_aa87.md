- **JS SPA** (no server-side text): stuff.co.nz, msn.

### Key learning — subscription vs block
NYT is NOT blocked: it's a **soft paywall** (full text in initial HTML, paywall is a JS overlay), so the JS-off fetch importer reads it fine *without* any subscription — already imported. A subscription (cookie auth) only helps **401** cases, NOT **403** bot-blocks (refused before auth). Most failures here are 403 on free sites, where a subscription does nothing. Real fix for 403+SPA+paywall is a logged-in headless browser (Playwright / browser39).
